¿QUÉ ES HOMESAVVY?

Somos una aplicación web cuyo propósito es facilitar a las personas la búsqueda y oferta de servicios relacionados con trabajos del hogar. Actuamos como un intermediario que facilita la conexión entre aquellos que necesitan servicios para el hogar y aquellos que pueden proporcionarlos.
